# 🏠 Housify - Real Estate Platform

A real estate website for buying, selling, and renting properties.

## 📁 Project Structure

- `housify-frontend/` - HTML, CSS, JavaScript files
- `housify-backend/` - Node.js backend with MongoDB

## 🚀 Features

- User registration and login
- Browse properties
- Add new properties
- Search and filter
- User dashboard

## 🛠️ Technologies Used

**Frontend:**
- HTML5
- CSS3
- JavaScript

**Backend:**
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## 📦 How to Run

### Backend:
```bash
cd housify-backend
npm install
npm run dev
